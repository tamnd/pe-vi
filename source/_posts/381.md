# [Problem 381](https://projecteuler.net/problem=381)
***
**(prime-k) factorial**

For a prime p let S(p) = (∑(p-k)!) mod(p) for 1 ≤ k ≤ 5.

For example, if p=7,
(7-1)! + (7-2)! + (7-3)! + (7-4)! + (7-5)! = 6! + 5! + 4! + 3! + 2! = 720+120+24+6+2 = 872. 
As 872 mod(7) = 4, S(7) = 4.

It can be verified that ∑S(p) = 480 for 5 ≤ p &lt; 100.

Find ∑S(p) for 5 ≤ p &lt; 10<sup>8</sup>.
***
## **Giai thừa (nguyên tố trừ k)**

$S(p) = \sum_{n=1}^{5} (p-n)! \bmod p$ với $p$ nguyên tố.

Tính $\sum_{5 \le p < 10^8} S(p)$.
