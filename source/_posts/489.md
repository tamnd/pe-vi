**Common factors between two sequences**

Let G(a, b) be the smallest non-negative integer n for which <dfn title="Greatest common divisor">gcd</dfn>(n<sup>3</sup> + b, (n + a)<sup>3</sup> + b) is maximized.
For example, G(1, 1) = 5 because gcd(n<sup>3</sup> + 1, (n + 1)<sup>3</sup> + 1) reaches its maximum value of 7 for n = 5, and is smaller for 0 ≤ n &lt; 5.
Let H(m, n) = Σ G(a, b) for 1 ≤ a ≤ m, 1 ≤ b ≤ n.
You are given H(5, 5) = 128878 and H(10, 10) = 32936544.

Find H(18, 1900).
***
## **Thừa số chung giữa hai dãy**

$\gcd(a_n, b_n)$ với $a_n = n!+1$ và $b_n = (n+1)!-1$. Tính $\sum_{n=1}^{10^6} \gcd(a_n, b_n)$.
