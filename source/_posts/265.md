**Binary Circles**

2<sup>N</sup> binary digits can be placed in a circle so that all the N-digit clockwise subsequences are distinct.

For N=3, two such circular arrangements are possible, ignoring rotations:
<center><img src="/resources/images/0265_BinaryCircles.gif"></center>

For the first arrangement, the 3-digit subsequences, in clockwise order, are: 000, 001, 010, 101, 011, 111, 110 and 100.

Each circular arrangement can be encoded as a number by concatenating the binary digits starting with the subsequence of all zeros as the most significant bits and proceeding clockwise. The two arrangements for N=3 are thus represented as 23 and 29:

<center>00010111&thinsp;<sub>2</sub> = 23
00011101&thinsp;<sub>2</sub> = 29</center>

Calling S(N) the sum of the unique numeric representations, we can see that S(3) = 23 + 29 = 52.

Find S(5).

## **Vòng tròn nhị phân**

$2^N$ chữ số nhị phân xếp vòng tròn sao cho tất cả chuỗi con $N$ chữ số chiều kim đồng hồ đều phân biệt.

Với $N = 5$, tính tổng các số nguyên $16$ chữ số nhị phân biểu diễn các vòng tròn nhị phân hợp lệ (bỏ qua phép quay).
